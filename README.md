# Cloud9 Workspace

A modern, responsive landing page for Cloud9 Workspace, a premium coworking space located in Alexandria, Egypt.

## Overview

Cloud9 Workspace is designed to be a comfortable and inspiring place for students, freelancers, entrepreneurs, and remote workers to work, study, collaborate, and grow. This landing page showcases the workspace's amenities, pricing, gallery, and community lifestyle.

## Features

*   **Responsive Design:** Fully responsive layout that looks great on desktops, tablets, and mobile devices.
*   **Modern Aesthetics:** Clean, premium design utilizing the Plus Jakarta Sans font and beautiful imagery.
*   **Smooth Animations:** Engaging scroll animations powered by GSAP (GreenSock Animation Platform) and ScrollTrigger.
*   **Interactive Galleries:** Image galleries for the workspace and café menu using GLightbox.
*   **Sections Included:**
    *   **Hero:** Catchy introduction with background imagery.
    *   **About:** Workspace overview and key statistics.
    *   **Services:** Detailed amenities (Fast Wi-Fi, Pro Workspaces, Meeting Rooms, etc.) with pricing information.
    *   **Why Choose Cloud9:** Bento-grid style feature highlights.
    *   **Tour:** Video tour section.
    *   **Café Menu:** Signature drinks and full menu gallery.
    *   **Gallery:** High-quality images of the workspace.
    *   **Community / Lifestyle:** Showcasing the relaxing and networking aspects of the space.
    *   **Testimonials:** Reviews from the community.
    *   **FAQ:** Accordion-style frequently asked questions.
    *   **Contact:** Phone, WhatsApp, Social Media links, and a Google Maps embed.

## Technologies Used

*   **HTML5**
*   **CSS3:** Custom styles, animations, and responsive utilities.
*   **JavaScript:** Vanilla JS for logic, mobile menu, and initializing libraries.
*   **Libraries:**
    *   [GSAP](https://greensock.com/gsap/): For high-performance animations.
    *   [GLightbox](https://biati-digital.github.io/glightbox/): For responsive image/video lightboxes.
    *   [Lucide Icons](https://lucide.dev/): For clean SVG icons.
    *   [FontAwesome](https://fontawesome.com/): For social media icons.
